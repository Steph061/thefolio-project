# BTS Blog Assets

This folder should contain the following images for the React blog:

## HomePage Images:
- `bts-hero.jpg` - Hero image (800x400px recommended)
- `album1.jpg` to `album10.jpg` - Album cover images (200x200px recommended)

## AboutPage Images:
- `bts-group.jpg` - BTS group photo (500x500px recommended)
- `bts-stage.jpg` - BTS performing on stage (500x500px recommended)

## ContactPage Images:
- `map-placeholder.jpg` - Map of Seoul, South Korea (600x400px recommended)

## Current Setup:
Currently using Lorem Picsum placeholder images for demonstration.
Replace the placeholder URLs in the components with local paths when you add actual images:

Example:
- Change: `src="https://picsum.photos/800/400?random=1"`
- To: `src="/assets/bts-hero.jpg"`

All images should be optimized for web (compressed) and properly licensed for use.
